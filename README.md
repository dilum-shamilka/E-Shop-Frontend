🚀 Tech Stack
Framework: ⚛️ React 18
Language: 🧠 TypeScript
Build Tool: ⚡ Vite
Styling: 🎨 Tailwind CSS
Data Fetching: 🔄 Axios
State Management: 🗂 Zustand
Routing: 🌐 React Router DOM
Deployment: ☁️ Vercel
📦 Features
Authentication: Secure User login and signup.
Product Discovery: Listing, searching, and filtering products.
Shopping Experience: Integrated Add to Cart and Checkout flow.
Management: Order summary and status tracking.
UI/UX: Fully responsive design with Toast notifications for actions.
Payments: COD system
